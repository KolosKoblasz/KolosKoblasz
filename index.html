Online report parser for Xilinx Vivado build reports

This site parses your FPGA project's report files

First build your project successfuly!
Upload your Vivado project's interesting files
(the bold parts must not be changed in the file names in order to successfully parse the report files):

  • "YourProjectFolder/project.runs/impl_n/top_utilization_placed.rpt"
  • "YourProjectFolder/project.runs/impl_n/top_timing_summary_routed.rpt"
  • "YourProjectFolder/project.runs/impl_n/top_utilization_placed_hierarchical.rpt"
    • Hierarchical report file can be created by the following Vivado TCL command (modify the command according to your case):
    "report_utilization -hierarchical -file ./YourProjectFolder/project.runs/impl_n/top_utilization_placed_hierarchical.rpt"
  • Etc...
Currently only utilization files are parsed and visualized
